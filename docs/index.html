Questions

Is it really free?

Yes. ScrollPay is free to install and use. You earn XP just by browsing with it on.

How do I win a prize?

Every XP you earn is one entry in the weekly draw. We pick a winner each week and contact them at the email you provide. The more XP you've earned, the more entries you have.

What data do you collect?

Only what's needed to count ad views and run the draw: the domain you're on (e.g. "twitter.com"), whether an ad was visible, whether you tapped it, and an optional email so we can reach you if you win. Never page content, URLs, or anything you type. See our privacy policy.

Which sites does it work on?

Everywhere — Twitter/X, Reddit, YouTube, Instagram, TikTok, and any other site you visit.
